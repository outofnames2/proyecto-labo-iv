# Un Trabajo por Hacer

## Dependencias/Correr la pagina

``` shell
yarn install

yarn watch
```

## Generar css (dev y release)
``` shell
yarn dev-tw

yarn prod-tw

```

## Re-generar css onchange (dev y release)
``` shell
yarn watch:dev

yarn watch:prod
```
## Borrar tooh

``` shell
yarn clean
```

## Release-build

``` shell
yarn release
```
